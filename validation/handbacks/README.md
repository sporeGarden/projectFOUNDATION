# validation/handbacks/

Geological record from projectNUCLEUS deployment validation sessions.

These documents capture capability gaps, security findings, and audit
observations discovered during live primal composition runs. Each handback
is a historical snapshot — counts and status reflect their date of creation,
not current reality.

**For current validation status**, see `validation/README.md`.

## Archive

9 handbacks from May 6–16, 2026 have been moved to `archive/`.
They remain available as fossil record but contain stale counts
(e.g., 184 targets, 30 workloads) that predate current manifests.
